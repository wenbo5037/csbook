# Csbook
### 阶段详解
##### 01-基础
该阶段主要包含：
1. 常识补充(网络应用架构、抓包、命令、算法逆向和密码学基础)
2. 信息打点(信息收集的一些方法)
3. 安全开发入门(包含但不限于html,css,js,php(这四个无论如何都得会，至少看得懂代码，查资料能知道代码要做什么),java,python)
这里做一个说明：只是学web渗透，php和java能够覆盖80%的网站场景，但是最好学习一下其他语言的基本知识，比如golang
c/c++以及汇编、安卓开发、ios开发、鸿蒙开发主要在逆向那里讲到
##### 02-web安全
该阶段主要包含的是常见的web漏洞(基本上只要是出现过的我都会写)
